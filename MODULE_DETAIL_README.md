# Детальная страница модуля - ПОЛНАЯ ВЕРСИЯ

## Описание

Создана полная функциональность для детального просмотра модулей с красивым дизайном, возможностью редактирования содержимого и добавления курсов. Включает левый сайдбар как у всей платформы.

## Новые модели

### ModuleSection
- Секции модуля (теория, практика)
- Поля: module, section_type, title, description, order

### ModuleSkill
- Навыки модуля (Listening, Writing, Grammar и т.д.)
- Поля: module, skill_type, title, description, is_available, order

### TheoryCard
- Теоретические карточки
- Поля: section, title, content, order

### ModuleAttachment
- Прикрепленные файлы к модулю
- Поля: module, file_type, title, file, description, file_size, order

## Новые URL маршруты

- `/modules/<id>/detail/` - Детальная страница модуля
- `/modules/<id>/edit/` - Редактирование содержимого модуля
- `/modules/<id>/add-section/` - Добавление секции
- `/modules/<id>/add-skill/` - Добавление навыка
- `/modules/<id>/add-attachment/` - Добавление файла
- `/modules/<id>/section/<section_id>/add-card/` - Добавление карточки к секции

## Использование

1. Перейдите на страницу модуля: `/modules/<id>/`
2. Нажмите кнопку "Пройти модуль"
3. Вы попадете на детальную страницу: `/modules/<id>/detail/`
4. Для редактирования нажмите "Редактировать содержимое"
5. Для добавления элементов используйте кнопки "Добавить..." в разделе редактирования
