# Руководство по созданию заданий грамматики

## Обзор

Новый интерфейс создания заданий предоставляет удобный и интуитивный способ создания различных типов грамматических заданий для студентов.

## Типы заданий

### 1. Соединение слов (Matching)
**Описание:** Студенты соединяют английские фразы с их русскими переводами.

**Как создать:**
1. Выберите тип "Соединение слов"
2. Заполните основные поля (название, описание, инструкция)
3. В левом поле введите английские фразы (по одной на строку)
4. В правом поле введите русские переводы (в том же порядке)
5. Используйте кнопку "Заполнить примером" для быстрого заполнения

**Пример:**
```
Английский:
I work
She is reading
We will go

Русский:
Я работаю
Она читает
Мы пойдем
```

### 2. Переписывание предложений (Sentence Rewrite)
**Описание:** Студенты переписывают предложения с правильной формой глагола.

**Как создать:**
1. Выберите тип "Переписывание предложений"
2. Заполните основные поля
3. В поле "Пример предложения" укажите, что нужно переписать
4. В поле "Правильный ответ" укажите правильный вариант

**Пример:**
```
Пример: She (go) to school every day → Present Simple
Ответ: She goes to school every day
```

### 3. Выбор перевода (Translation Choice)
**Описание:** Студенты выбирают правильный перевод из предложенных вариантов.

**Как создать:**
1. Выберите тип "Выбор перевода"
2. Заполните основные поля
3. Введите предложение на английском
4. Введите варианты перевода (по одному на строку)
5. Правильный вариант отметьте звездочкой (*) в начале

**Пример:**
```
Предложение: I have been studying English for 3 years
Варианты:
*Я изучаю английский уже 3 года
Я буду изучать английский 3 года
Я изучал английский 3 года
```

### 4. Составление предложений (Sentence Builder)
**Описание:** Студенты составляют предложения из предложенных слов.

**Как создать:**
1. Выберите тип "Составление предложений"
2. Заполните основные поля
3. Введите слова через запятую (включая знаки препинания)
4. Укажите правильное предложение

**Пример:**
```
Слова: She, is, reading, a, book, now
Правильное предложение: She is reading a book now
```

## Функции интерфейса

### Предварительный просмотр
- Нажмите кнопку "Предварительный просмотр" для просмотра задания
- Увидите, как задание будет выглядеть для студентов
- Проверьте правильность всех данных

### Кнопки "Заполнить примером"
- Быстро заполняют поля типичными примерами
- Помогают понять формат данных
- Можно использовать как основу для своих заданий

### Валидация
- Все обязательные поля помечены звездочкой (*)
- Проверка корректности данных
- Предупреждения об ошибках

## Советы по созданию эффективных заданий

### 1. Название задания
- Используйте понятные и описательные названия
- Указывайте тему или грамматическое правило
- Пример: "Соединение слов - Времена глаголов"

### 2. Описание
- Кратко объясните, что нужно сделать
- Укажите цель задания
- Пример: "Соедините английские фразы с их русскими переводами"

### 3. Инструкция
- Дайте пошаговые указания
- Объясните, как выполнить задание
- Укажите, что делать в случае ошибки

### 4. Содержание
- Используйте разнообразные примеры
- Убедитесь, что все варианты корректны
- Проверьте соответствие сложности уровню студентов

## Редактирование заданий

### Основная информация
- Измените название, описание, инструкцию
- Настройте порядок и статус задания
- Включите/выключите задание

### JSON содержимое
- Прямое редактирование JSON структуры
- Валидация синтаксиса JSON
- Предупреждения об ошибках

### Удаление
- Страница подтверждения удаления
- Предупреждение о необратимости действия
- Возможность отменить операцию

## Лучшие практики

1. **Тестируйте задания** - всегда проверяйте задания перед публикацией
2. **Используйте примеры** - кнопки "Заполнить примером" помогают понять формат
3. **Проверяйте предварительный просмотр** - убедитесь, что задание выглядит правильно
4. **Создавайте разнообразные задания** - используйте все типы заданий для лучшего обучения
5. **Организуйте по порядку** - используйте поле "Порядок" для логической последовательности

## Устранение неполадок

### Проблемы с JSON
- Проверьте синтаксис JSON
- Убедитесь в правильности кавычек и запятых
- Используйте валидатор JSON

### Проблемы с отображением
- Проверьте предварительный просмотр
- Убедитесь в корректности данных
- Проверьте порядок элементов

### Проблемы с сохранением
- Убедитесь, что все обязательные поля заполнены
- Проверьте права доступа
- Обновите страницу и попробуйте снова
